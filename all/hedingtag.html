heading tag

Main heading:

<h1> represents the highest level of heading and should be used to define the main title of the page. It is the most prominent and important heading on the page. Only one <h1> should be used per page, and it should describe the main topic or purpose of the page.

Here are example and result:

Code for explination:

<!DOCTYPE HTML>
<html>
<head>
<title>Main Heading Example</title>
</head>
<body>
<h1>Welcome to My Website</h1>
</body>
</html>

This is result:

Welcome to my Website

Sub heading:

<h2> through <h6> represent decreasing levels of subheadings, where <h2> is the second most important heading, <h3> is a subheading of <h2>, and so on. Use these tags to create a logical hierarchy of section headings and subheadings in your content.

Here are example and result:

Code for explination:

<!DOCTYPE HTML>
<html>
<head>
<title>Main Heading Example</title>
</head>
<body>
<h1>Article Title</h1>
<h2>Section 1: Introduction</h2>
<h3>Section 2: Methods</h3>
<h4>Subsection: Data Collection</h4>
<h5>Section 3: Conclusion</h5>
<h6>Section 4: the end</h6>
</body>
</html>

This is result:

Article Title


Section 1: Introduction


Section 2: Methods


Subsection: Data Collection


Section 3: Conclusion

section: the end
