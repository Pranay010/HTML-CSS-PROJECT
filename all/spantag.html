Span tag

The `<span>` tag in HTML is an inline container used to apply styles to small portions of text or elements within a larger block of content. Unlike block-level elements like `<div>`, the `<span>` tag does not add line breaks or create a new line; it is meant to style and group inline content.


Uses:

Code for explination:

<!DOCTYPE html>
<html>
<head>
<title>Span Tag Example</title>
</head>
<body>
<p>This is a <span style="color: blue;">blue</span> text.</p>
<p><span style="font-weight: bold;">This</span> is a <span style="font-style: italic;">styled</span> text.</p>
</body>
</html>

This is result:

Span Tag Example

This is a blue text.

This is a styled text.

Explanation:


In the above example, we have an HTML document with two paragraphs (`<p>`). Inside the paragraphs, we use the `<span>` tag to apply styles to specific portions of the text. In the first paragraph, we use the `<span>` tag with the `style` attribute to set the color of the word "blue" to blue. The `<span>` tag acts as a wrapper around the word, allowing us to apply the style only to that specific part of the text. In the second paragraph, we use two `<span>` tags. The first `<span>` tag sets the font weight to bold for the word "This," and the second `<span>` tag sets the font style to italic for the word "styled." By using multiple `<span>` tags, we can independently style different parts of the text within the same paragraph.

The `<span>` tag itself does not have any default styling, and its primary purpose is to provide a way to apply inline styles or target specific parts of text for CSS manipulation or JavaScript interactions. It is common to use the `<span>` tag when you want to style only a few words or characters within a larger block of text, rather than styling the entire paragraph. The use of classes and IDs is often preferred for more extensive or complex styling, as it helps keep the HTML code cleaner and allows for more reusability.


Example with CSS class:

Code for explination:

<!DOCTYPE html>
<html>
<head>
<title>Span Tag with CSS Class Example</title>
<style>
.highlight {
background-color: yellow;
color: red; }
</style>
</head>
<body>
<p>This is a <span class="highlight">highlighted</span>text.</p>
</body>
</html>

This is result:

Span Tag with CSS Class Example

This is a highlighted text.

In this example, we use a CSS class `.highlight` to set the background color to yellow for the word "highlighted." The `<span>` tag has a `class` attribute that links it to the CSS class, allowing us to apply the specified style to the text. Using CSS classes enables better separation of concerns and promotes more maintainable and reusable code.
