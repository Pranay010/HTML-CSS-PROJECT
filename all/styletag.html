Style tag

The <style> tag in HTML is used to define CSS (Cascading Style Sheets) rules directly within an HTML document. It allows you to apply custom styles to specific HTML elements without the need for an external CSS file. The styles defined inside the <style> tag apply only to the HTML document in which it is placed.

Usage:

Code for explination:

<!DOCTYPE html>
<html>
<head>
<title>Style Tag Example</title>
<style>
/* CSS rules go here */
body {
font-family: Arial, sans-serif;
}
h1 {
color: blue;
}
p {
font-size: 16px;
}
</style>
</head>
<body>
<h1>Welcome to my website</h1>
<p>This is a paragraph with custom styles.</p>
</body>
</html>

This is result:

Style Tag Example

Welcome to my website

This is a paragraph with custom styles.

Explanation:

In the above example, we have an HTML document with a <style> tag inside the <head> section. The <style> tag is used to define custom CSS rules that apply to the elements in the HTML document.

Inside the <style> tag, we define three CSS rules:

1. The body rule sets the font-family property to Arial (or sans-serif if Arial is not available) for all text within the <body> element.

2. The h1 rule sets the color property to blue for all <h1> elements in the document.

3. The p rule sets the font-size property to 16 pixels for all <p> (paragraph) elements in the document.


By using the <style> tag, we can directly embed CSS rules in the HTML document, providing a convenient way to apply simple styles without the need for an external CSS file.

Uses of the <style> tag:

1. Simple Styling: The <style> tag is useful for applying quick and straightforward styles to individual HTML documents without the need to create and link an external CSS file.

2. Specific Page Styling:When you want to style a single HTML document differently from the rest of the website, you can use the <style> tag to define custom styles for that specific page.

3. Responsive Styles: You can use the <style> tag in combination with media queries to apply responsive styles that adapt to different screen sizes.

4. Quick Experiments:The <style> tag is handy for testing and experimenting with CSS rules before adding them to an external CSS file.


However, for larger projects or websites with multiple pages, it is more efficient and organized to use an external CSS file. External stylesheets can be reused across multiple pages, reducing duplication of code and making maintenance easier.


To use external CSS, you can create a separate .css file and link it to your HTML documents using the <link> tag within the <head> section.

Code for explination:

<!DOCTYPE html>
<html>
<head>
<title>External CSS Example</title>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<!-- Content goes here -->
</body>
</html>

In this example, the styles.css file contains all the CSS rules that apply to the entire website, keeping the HTML documents clean and focused on the content.
