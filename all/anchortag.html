Anchor tag

The anchor tag in HTML is used to create hyperlinks, allowing users to navigate to different web pages or resources. When users click on a link created using the anchor tag, they are redirected to the URL specified in the href attribute of the anchor tag.


Usage:

code for explination

<!DOCTYPE HTML>
<html>
<head>
<title>Anchor Tag Example</title>
</head>
<body>
<p>click <a href="https://www.example.com"> </a>to visit example.com </p>
</body>
</html>

result

Explination:

In the above example, we have an HTML document with a paragraph <p> containing an anchor tag <a>. The anchor tag is used to create a hyperlink to the website "https://www.example.com". When the link is clicked, the user's web browser will open the specified URL, and they will be taken to the Example.com website.

Attributes of the anchor tag:

href: This is the most important attribute of the anchor tag. It specifies the destination URL that the link will point to. It can be an absolute URL (e.g., "https://www.example.com") or a relative URL (e.g., "/page.html"). target: This optional attribute specifies where the linked URL should be opened. Common values include "_blank" (to open the link in a new tab or window) and "_self" (to open the link in the same tab or window, which is the default behavior).

Example of using the target attribute:

code for explination

<!DOCTYPE HTML>
<html>
lt;head>
<title>Anchor Tag Example</title>
</head>
<body>
<p>click <a href="https://www.example.com" target="_blank> </a>to visit example.com </p>
</body>
</html>
