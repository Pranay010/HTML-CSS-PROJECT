List tag

In HTML, there are two main types of list tags: unordered lists (<ul>) and ordered lists (<ol>). Lists are used to organize and present related pieces of information in a structured manner. Each list item is represented by the (<li>) tag, which stands for "list item." Here's an explanation of how to use list tags with some example code:

1. Unordered List (<ul>):

Theory:

(<ul>)represents an unordered list, which means the items in the list do not have a specific order or sequence. These lists are typically styled with bullet points or other markers to denote individual list items.

Uses:

Unordered lists are suitable for presenting items without any particular numerical or chronological order.


Example code:

Code for explination:

<!DOCTYPE HTML>
<html>
<head>
<title>Unordered List Example</title>
</head>
<body>
<h1>Fruits</h1>
<ul>
<li>apple<li>
<li>Orange<li>
<li>Banana<li>
<li>Strawberry<li>
</ul>
</body>
</html>

This is result:

Unordered List Example

Fruits

  • Apple
  • Orange
  • Banana
  • Strawberry


2. Ordered List<ul>:

Theory:

(<ol>)represents an ordered list, where the list items have a specific numerical or alphabetical order. These lists are typically styled with numbers or letters to denote the sequence of items.

Use:

Ordered lists are useful when presenting information in a specific order, such as steps in a process or ranking items.

Example code:

Code for explination:

<!DOCTYPE HTML>
<html>
<head>
<title>Unordered List Example</title>
</head>
<body>
<h1>Fruits</h1>
<ol>
<li>apple<li>
<li>Orange<li>
<li>Banana<li>
<li>Strawberry<li>
</ol>
</body>
</html>

This is result:

Unordered List Example

Fruits

  1. Apple
  2. Orange
  3. Banana
  4. Strawberry


Nested Lists:

Both unordered and ordered lists can be nested inside each other to create hierarchical lists. For example, you can have an ordered list containing multiple unordered lists, or vice versa.


Code for explination:

<!DOCTYPE HTML>
<html>
<head>
<title>Unordered List Example</title>
</head>
<body>
<h1>Shopping List</h1>
<ul>
<li>Fruits<li>
<ul>
<li>apple<li>
<li>Oranges<li>
</ul>
</li>
<li>Vegetables<li>
<ul>
<li>Carrots<li>
<li>Broccoli<li>
</ul>
</li>
</ul> </body>
</html>

This is result:

Nested List Example

Shopping List

  • Fruits
    • Apples
    • Oranges
  • Vegetables
    • Carrots
    • Broccoli
